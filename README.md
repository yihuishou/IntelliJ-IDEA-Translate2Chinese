# IntelliJ-IDEA-Translate2Chinese
## 基于IntelliJ IDEA 2017.1.5 的汉化项目

## 本汉化补丁只适配于IntelliJ IDEA 2017.1.5对其他版本兼容性未知

### 使用方法（一）

1. 请自行备份原版语言包`/lib/resources_en.jar`
2. 用任意解压工具打开原版语言包`resources_en.jar`
3. 复制本项目中`resources_cn`中的所有文件 
4. 覆盖到打开的原版语言包中
5. 保存后重启IDEA即可

>PS：执行上述操作时请不要打开IDEA
